🚀 How to Use

Clone the repo:
git clone https://github.com/omgajipara7/Data-Science.git
Navigate to your project folder:
cd Data-Science
Install dependencies (if required):
pip install -r requirements.txt
Run the Jupyter Notebook:
jupyter notebook
